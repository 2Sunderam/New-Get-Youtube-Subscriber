This is the Server for getting the data from the database

Fetch all subscribers from the database, selecting only the 'name' field using: https://subscribers-oyrz.onrender.com/subscribers

Fetch all subscribers from the database, selecting only the 'name' and 'subscribedChannel' fields using: https://subscribers-oyrz.onrender.com/subscribers/names

Find a single subscriber in the database based on the provided name parameter: like
  • https://subscribers-oyrz.onrender.com/subscribers/John Doe
  • https://subscribers-oyrz.onrender.com/subscribers/Jeread Krus
  • https://subscribers-oyrz.onrender.com/subscribers/Lucifer
